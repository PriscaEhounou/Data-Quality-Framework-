# Data Quality Framework - Mortgage Dataset

## Overview
This project is a Python-based data quality checker designed to validate a mortgage dataset.

It identifies common data issues such as missing values, duplicates, and invalid data.

## Features
- Detect missing values
- Detect duplicate rows
- Validate Loan Amount (> 0)
- Validate Interest Rate (0–15%)
- Detect missing borrower names
- Validate Loan Status
- Generate Data Quality Score
- Generate text report

## Tools Used
- Python
- pandas

## Project Structure
- Data_Quality_Checker.py
- mortgage_data.csv
- Output/quality_metrics_report.txt
